# ProjectPortals

Pjp is a plugin aiming to create nice and customisable portals.
Thus it requires a bit of attention, you will quickly be able to connect lots of gates through your worlds.

For more information, read the wiki : https://github.com/trentech/ProjectPortals/wiki
There is also a dedicated sponge topic : https://forums.spongepowered.org/t/project-portals/10438
